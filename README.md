# Baponi

Sandboxed code execution for AI agents. Billed per execution, zero idle cost: session state persists between calls while nothing runs. Mount your own S3, GCS, or Azure bucket as a local filesystem inside the sandbox.

**[Website](https://baponi.ai)** | **[Docs](https://baponi.ai/docs)** | **[Pricing](https://baponi.ai/pricing)** | **[Python SDK](https://github.com/baponi/baponi-sdk)**

```bash
pip install baponi
```

- Works with any MCP client: Claude Desktop, Cursor, Windsurf
- REST API and Python SDK, with LangChain, OpenAI, Anthropic, Gemini, and CrewAI integrations
- Cloud SaaS, or self-hosted in your own Kubernetes cluster from one Helm chart
